CSS简介

CSS（层叠样式表）是一种样式表语言，用于描述HTML文档的呈现方式。CSS使开发者能够将内容与设计分离，从而更高效地创建视觉吸引力强的网页。

style.css
body {
  background-color: #f0f0f0;
  font-family: 'Arial', sans-serif;
}

.container {
  width: 80%;
  margin: 0 auto;
  padding: 20px;
}

CSS选择器

选择器用于定位HTML文档中的元素，以便应用样式。了解不同类型的选择器及其优先级是掌握CSS的关键。

常用选择器类型

selectors.css
/* 元素选择器 */
p { color: blue; }

/* 类选择器 */
.highlight { background: yellow; }

/* ID选择器 */
#header { font-size: 2rem; }

/* 属性选择器 */
a[target="_blank"] { color: red; }

/* 伪类选择器 */
button:hover { background: #555; }
元素选择器：<p>
类选择器：.highlight
ID选择器：#header

盒模型

CSS盒模型是网页布局的基础概念。每个元素都被视为一个矩形盒子，由内容(content)、内边距(padding)、边框(border)和外边距(margin)组成。

box-model.css
.box {
  width: 200px;
  height: 150px;
  padding: 25px;
  border: 15px solid #FFB6C1;
  margin: 30px;
  background-color: #FFD1DC;
}

重要提示

使用box-sizing: border-box;可以让元素的宽度和高度包含内边距和边框，使布局计算更直观。

Flexbox布局

Flexbox是一种一维布局模型，用于在行或列中排列元素。它非常适合创建响应式布局，使元素能够自适应容器。

flexbox.css
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.item {
  flex: 1;
  margin: 10px;
}

Flex属性速查

justify-content: 主轴对齐方式
(flex-start, center, flex-end, space-between, space-around)
align-items: 交叉轴对齐方式
(flex-start, center, flex-end, stretch, baseline)
flex-direction: 主轴方向
(row, column, row-reverse, column-reverse)

CSS Grid布局

CSS Grid是一个二维布局系统，可以同时处理行和列。它非常适合创建复杂的网页布局。

grid.css
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-gap: 15px;
}

.header {
  grid-column: 1 / -1;
}

.sidebar {
  grid-row: 2 / 4;
}

响应式设计

响应式设计确保网页在各种设备上都能良好显示。主要技术包括媒体查询、流式布局和弹性图片。

responsive.css
/* 移动优先设计 */
.container {
  padding: 15px;
}

/* 平板设备 */
@media (min-width: 768px) {
  .container {
    padding: 30px;
    max-width: 720px;
  }
}

/* 桌面设备 */
@media (min-width: 992px) {
  .container {
    max-width: 960px;
  }
}

不同设备上的布局变化

响应式设计原则

1. 移动优先：先设计移动端布局，再逐步增强大屏幕体验

2. 流式网格：使用百分比而非固定宽度

3. 弹性图片：设置max-width: 100%防止图片溢出

4. 媒体查询：根据设备特性应用不同样式
